# Startup

A single-player engine-building card game built in Phaser 3. Place personnel cards into rows, activate rows to trigger effects, and build toward a valuation across four rounds.

## Running locally

Requires a local server (browser security blocks file:// loading of JSON).

```
npx serve
```

Then open the URL shown in your terminal.

## Tech

- [Phaser 3](https://phaser.io/) via CDN
- Vanilla JavaScript — no build step
- Card data loaded at runtime from `cards.json`
